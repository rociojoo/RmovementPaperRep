README file for Import and ImportSuggest csv files:

RmovePackages-Import.csv:
It contains in one column the name of a package and on the second column the package they import, depend on, or link to.

RmovePackages-ImportSuggest.csv:
It contains in one column the name of a package and on the second column the package they import, depend on, link to or suggest.

This information was updated on August 30th 2018.
